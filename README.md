使用說明：

1. 完成線上表格填寫後，將表格下載為 Excel 檔。

2. 將下載的 Excel 檔與 `convert_to_nani.py` 放置於同一個資料夾中。

3. 執行 `convert_to_nani.py`，將 Excel 表格內容轉換為 NaniScript 檔案。

4. 將轉換後的 NaniScript 檔案匯入至具備 Naninovel 套件的專案中驗證。
